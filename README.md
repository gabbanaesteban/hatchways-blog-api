# Hatchways Backend Assessment

## Requirements

Node >= 14

## Setup

``` bash
npm ci
```

``` bash
npm start
```

## Basic Usage

```bach
curl 'http://localhost:3001/api/ping'
```

## Testing

You can run the tests with:

```bash
npm run test:coverage
```
